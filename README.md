# n8n Competitors News Monitoring Workflow

## Description
Weekly automated competitor monitoring using Google Sheets, OpenAI, and Perplexity.

## Prerequisites
- n8n instance
- Google Sheets API credentials
- OpenAI API key
- Perplexity API key
- Gmail account

## Setup
1. Import this workflow in n8n
2. Replace all `YOUR_*_CREDENTIAL_ID` with your actual credentials
3. Update `YOUR_GOOGLE_SHEET_ID` with your sheet ID
4. Change `your.email@example.com` to your email
5. Activate the workflow

## Google Sheet Format
Create a sheet with a column named "Competitor" containing competitor names.
